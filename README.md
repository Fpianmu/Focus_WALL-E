# 从容应队WALL-E桌宠使用说明
## 模式简介
### MODE_00： 唤醒与休眠
唤醒词：<br>
唤醒：瓦力瓦力<br>
休眠：退下  歇着吧<br>
### MODE_01：唱歌
唤醒词:唱首歌  唱歌模式<br>
可选歌曲:<br>
* 《我怀念的》
*  《晴天》
*  《库里之歌》
### MODE_02：剪刀石头布
唤醒词：剪刀石头布  石头剪刀布<br>
交互词：剪刀/剪子/石头/锤头/锤子/布<br>
退出模式：不玩了/退出剪刀石头布<br>
### MODE_03：萝卜纸巾
唤醒词：大开门模式/萝卜纸巾<br>
交互词：萝卜/胡萝卜/纸巾/卫生纸/真棒<br>
当瓦力听到真棒后退出该模式<br>
### MODE_04：自动跟随　
唤醒词：跟我走/自动跟随<br>
退出模式：其他任何交互词均可退出<br>
### MODE_05：手势识别
唤醒词：手势识别\看我手势\稍息立正<br>
交互：用户发出的上下、左右、顺逆时针手势<br>
退出模式：其他任何交互词均可退出<br>
## 常见错误及解决方案
### 情况一：
#### 错误：<br>
发出指令后没有回复<br>
#### 解决办法：<br>
1. 重新说一遍指令<br>
1. 检查是否发出口令属于该模式<br>
1. 检查口令是否是规定的命令词<br>
1. 检查周围环境是否嘈杂<br>
### 情况二：
#### 错误：<br>
发出指令有回复但是没有动作<br>
#### 解决办法：<br>
1.回到“瓦力瓦力”重新发令<br>
1.关机重启(reset)(doge)<br>
